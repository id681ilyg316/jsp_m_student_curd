## 本项目实现的最终作用是基于JSP简单学生信息管理系统带分页
## 分为1个角色
### 第1个角色为管理员角色，实现了如下功能：
 - 分页功能
 - 增加学生
 - 学生列表查询
 - 更新学生
 - 管理员登录
